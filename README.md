Repositori ini berisi catatan perjalanan belajar saya dalam memahami **Transformer Architecture**.

Studi kasus yang digunakan adalah timeseries forceasting dan classification pada dataset kompetisi SSDS 2024, saya sebelumnya berencana untuk melakukan multitask learning di mana melakukan dua task tersebut secara bersamaan namun masih belum saya temukan cara yang terbaik untuk itu

Pada eksperimen ini saya menggunakan bagian Encodernya saja dari Transformer untuk melakukan regresi dan untuk klasifikasi saya menggunakan XGBoost dengan SMOTE untuk meningkatkan F1 score
